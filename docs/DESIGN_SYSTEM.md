# Design System UrbanTraffic

## Paleta principal

- Primary: `#4CAF50`
- Hover: `#45A049`
- Dark: `#2E7D32`
- Soft background: `#E8F5E9`
- Text dark: `#2C3E50`

## Tipografia

- Família principal: `Inter`
- Títulos: peso 600–700
- Corpo: peso 400–500

## Componentes

- Botões primários: fundo verde, texto branco, raio 8px
- Botões outline: contorno verde, hover preenchido
- Cards: fundo branco, sombra suave, cantos arredondados
- Badges: verde para ações positivas; verde suave para estados secundários
- Inputs: borda clara com foco verde

## Mapa

- Linha principal: verde escuro
- Caminhada: verde claro tracejado
- Paragem origem: verde
- Paragem destino: verde escuro
- Stops intermédios: verde suave
- Autocarro selecionado: verde escuro destacado

## Estados visuais

- Sucesso: verde principal
- Erro: vermelho Bootstrap
- Aviso: amarelo/âmbar Bootstrap
- Realtime: badge verde sólido
- Estimado: badge verde suave
- Fallback: badge verde suave
- Simulated: badge verde suave

## Regras de uso

- Evitar arco-íris em linhas e badges
- Preferir variações de verde antes de cores aleatórias
- Usar alto contraste em informação crítica
- Em páginas académicas, manter consistência visual com a app principal
